onConnectionLost

abstract fun onConnectionLost(ns: Socket, error: ErrorType, peerAddress: InetSocketAddress, token: Int): Unit

Called just after a pending connection in the background has been resolved and the connection has failed.

See Also: srt_connect_callback

Parameters

ns - the new SRT socket of the new connection

error - the reason connect has been lost

peerAddress - the address of the peer device

token - the token value, if it was used for group connection, otherwise -1

Return
return 0, if the connection is to be accepted. If you return -1, this will be understood as a request to reject the incoming connection.
